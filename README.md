# BDFW---DOCUMENTAÇÃO
Documentação da linguagem basica de criação de bots na plataforma whatsapp (bdfw).

# Enviar uma mensagem:
```js
$send();
```
#Exemplo de uso:
````js
$send("Hello World !");
````

#Responder a uma mensagem
```js
$reply("");
```

#Exemplo de uso:
```js
$reply("I replied to your message!");
```
